# 日常
今年的暑假因為疫情提早了許多開始放假，\
也因為疫情導致暑假特別的無聊，\
每天的日常也就是玩玩遊戲、寫寫程式，\
每天都只做這些也會有無聊的時候，\
也就只能找新的遊戲玩，或是找有興趣的東西研究，\
不過幸好7/27就解除了三級封鎖，\
雖然解除了但也不是絕對的安全，\
也就只能乖乖地在家，最多就出門買東西吃，\
期望疫苗可以趕快買到，全國人民都可以施打，\
讓疫情趕快消失，生活回到以前的模樣。
